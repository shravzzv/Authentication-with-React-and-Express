REST API for the "Authentication with React and Express" project.

It contains the following routes:

POST /signup
Requires a username (8-32 characters) and a password (min. 8 characters) in the body. If valid parameters are passed in the request body, returns a user object with a username field and a JWT token, or else returns the errors in a json format. The JWT contains the user document's id from the database as the payload guarded by a strong secret.
